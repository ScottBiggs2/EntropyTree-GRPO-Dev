# Archive

Historical design documents, implementation guides, and review notes from earlier
phases of the project (MDLM toy stack, initial Dream migration, early concept
reviews).

These files are preserved for reference but are **not** the current source of
truth for any active work. See the root `readme.md`, `research_decisions.md`,
`literature_reference.md`, and the `dream/` directory for current documentation.
